# Static LLM Capacity Profiles

R25H records the static decoder capacity envelope as project policy. These
profiles are not remote platform facts, model recommendations, or admission
records. They are local gates for future reviewed candidate decisions.

- `hobby_static_llm_lite` is a constrained fallback/rejection profile.
- `pro_static_llm_full` is the primary static profile for realistic browser
  decoder artifact review.

Capacity scenarios are metadata-only. They do not create model weights, do not
select a named model, and do not admit production assets.
