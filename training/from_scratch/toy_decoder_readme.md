# R25J Tiny Decoder Toy Pipeline

The tiny decoder toy is a pipeline-mechanics scaffold only. It is not the
product model, not a benchmark, not a browser release candidate, and not a
long-training path.

R25J does not run decoder training. The default overfit command exits with a
safe skip and writes no weights. Any future toy sanity run must write only to
ignored `artifacts/training_os/tiny_decoder_toy/` paths and must remain
explicitly labeled not product.

Formal training progress remains `0%` in R25J.
